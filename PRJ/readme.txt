[실험 환경]
OS : Windows 10 Pro 22H2, x64-based processor
Processor : AMD Ryzen 5 5600X 6-Core Processor 3.70GHz, NVIDIA GeForce RTX 3070 Ti 8GB
IDE : Visual Studio 2022, openFrameworks(C/C++)

[빌드 방법]
'openFrameworks 라이브러리를 포함할 수 있는 디렉토리 내'의 Visual Studio 2022의 솔루션에서 해당 프로그램을 릴리즈 모드로 빌드할 수 있음.
>> oF/apps/myApps/ 디렉토리 내에 솔루션 폴더(prj2)를 위치시키는 것을 추천
windows.h 헤더가 포함되어 있으므로 반드시 Windows 운영 체제에서 빌드해야 함.
빌드된 프로그램은 솔루션 디렉토리 내의 /bin/prj2.exe을 실행하면 됨. 사전에 빌드해둔 executable file이 존재하니 빌드하지 않고 실행 가능함.

*** 자세한 동작은 직접 프로그램을 실행해보시고, 보고서 및 발표 영상을 참고해주세요.
